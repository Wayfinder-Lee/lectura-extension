📖 LECTURA LLM 设置教程

以 DeepSeek 为例，国内可直接使用，注册即送免费额度

为什么选择 DeepSeek？

  • 🇨🇳 国内可直接访问，无需代理
  • 💰 注册送免费额度，足够日常查词使用
  • 📊 API 完全兼容 OpenAI 格式
  • ⚡ 响应速度快，适合即时查词场景

步骤 1：注册 DeepSeek 账号

  1. 打开 platform.deepseek.com
  2. 点击右上角 Sign Up 注册（支持手机号或邮箱）
  3. 完成注册后登录进入控制台

步骤 2：获取 API Key

  1. 登录后在左侧菜单点击 API Keys
  2. 点击 Create API Key 按钮
  3. 输入名称（如 "LECTURA"），点击创建
  4. ⚠️ 立即复制保存 API Key（只显示一次！）
API Key 格式：sk-xxxxxxxxxxxxxxxxxxxxxxxx
请妥善保管，不要分享给他人。

步骤 3：配置 LECTURA

  1. 打开 LECTURA 设置页面
  2. 展开 🤖 LLM API 设置
  3. LLM 提供商选择 DeepSeek（会自动填入端点和模型）
  4. 粘贴刚才复制的 API Key 到 API Key 输入框
  5. 保持默认模型 deepseek-v4-flash（速度快、成本低）
  6. 确保 自动使用 LLM 已勾选
推荐配置：
提供商：DeepSeek
端点：https://api.deepseek.com（自动）
模型：deepseek-v4-flash

📌 选 deepseek-v4-flash 即可，速度快且免费额度充足。

步骤 4：测试

配置完成后，在任意网页中选一个不常见的英文单词（如 serendipity），如果离线词典未收录，LECTURA 会自动调用 DeepSeek 生成释义。

查词结果会标注 🤖 LLM 标记。

常见问题

Q: API 调用会花多少钱？

DeepSeek flash 模型价格极低，单次查词约 0.001 元。注册赠送的额度足够数月日常使用。

Q: 查词速度怎么样？

DeepSeek flash 模型通常在 1-2 秒内返回结果。如果觉得慢，可以关闭自动 LLM 查询，改为手动触发。

Q: 为什么建议 deepseek-v4-flash？

flash 版本专为速度和成本优化，查词、翻译这类简单任务完全够用，不需要更贵的 chat 版本。

Q: Prompt 可以自定义吗？

可以！展开设置页面的 📝 Prompt 设置，你可以完全自定义查词和翻译的提示词格式。
